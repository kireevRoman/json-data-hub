# JSON Data Hub

Этот репозиторий служит удобным источником JSON-данных через GitHub Pages.  
Используйте URL вида:  
`https://kireevroman.github.io/json-data-hub/data/users.json`

## Доступные данные
- `/data/users.json` — тестовые пользователи
- `/data/products.json` — примеры товаров

## Для разработчика
- Ветка `main` защищена (только через PR)
- Все изменения данных — через Pull Request
